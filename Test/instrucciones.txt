1.- En la terminal ejecutar: npm install
2.- Crear un archivo ".env" y pegar:
DEFAULT_MESSAGE=true
SAVE_MEDIA=true
PORT=3000
DATABASE=none
LANGUAGE=es
SQL_HOST=
SQL_USER=
SQL_PASS=
SQL_DATABASE=
KEEP_DIALOG_FLOW=false
MULTI_DEVICE=true

3.- Entrar a node_modules->whatsapp-web.js->src->util->injected.js
y cambiar la linea 22 a:
window.Store.GroupMetadata = window.mR.findModule('GroupMetadata')[0].default.GroupMetadata;

4.- Para iniciar el chatbot ejecutar: npm run start
después de un momento aparecerá un codigo QR que se debe leer en whatsapp como si se
fuese a conectar un dispositivo nuevo, y eso es todo, ahora un bot responderá a los mensajes 

5.- No olvidar desvincular el bot en whatsapp al terminar
